# SGTI 傻狗测试

## 项目介绍
SGTI 傻狗测试是一个有趣的社交行为测试，通过回答30个问题，帮助你了解自己在社交中的行为模式和特点，最终将你归类为24种不同类型的"狗"之一。

## 测试内容

### 30个测试问题
测试包含30个关于社交行为的问题，涵盖了以下方面：
- 消息回复习惯
- 社交活动偏好
- 群聊行为
- 社交能量水平
- 朋友圈使用习惯
- 社交恐惧程度
- 社交方式偏好

### 24种狗种类型
根据测试结果，你将被归类为以下24种狗种之一：
1. 舔狗 - 秒回消息，过度热情
2. 潜水狗 - 默默观察，很少发言
3. 杠狗 - 喜欢反驳，爱抬杠
4. 冷场狗 - 一开口就冷场
5. 酒狗 - 喝酒后社交能力爆发
6. 吗喽狗 - 苦命打工人，社交能量低
7. 死狗 - 消息已读不回，佛系社交
8. 哈巴狗 - 习惯性讨好，不懂拒绝
9. 疯狗 - 互联网发疯文学代言人
10. 独狗 - 独来独往，喜欢独处
11. 舔而不得狗 - 舔狗进阶版，舔的人在舔别人
12. 电子狗 - 回复像客服，礼貌但冰冷
13. 土狗 - 网络用语落后，社交语言老旧
14. 流浪狗 - 社交边缘人，没有归属感
15. 跟屁狗 - 去哪都要人陪，害怕独处
16. 看门狗 - 只在小圈子活跃，大群沉默
17. 烧烤狗 - 社交价值≈推荐好吃的餐馆
18. 落水狗 - 朋友们的指定开玩笑对象
19. 失踪狗 - 从不主动联系，消息靠缘分回
20. 二哈 - 喜欢拆穿别人装逼，社交拆台专业户
21. 牧羊犬 - 喜欢组织聚会，掌控社交节奏
22. 警犬 - 喜欢收集他人信息，侦查能力强
23. 病狗 - 社交恐惧症晚期，看到消息就心慌
24. 神狗 - 一年出现一次，江湖有传说

## 如何使用

1. 回答测试中的30个问题，记录你的选择（A、B、C、D）
2. 根据答案计算每种狗种的得分
3. 得分最高的狗种即为你的测试结果
4. 查看对应狗种的详细解释和建议

## 结果解释
每个狗种结果都包含以下内容：
- 鉴定结果：你的社交行为类型
- 详细描述：你的典型社交行为特征
- 毒舌一句：对你社交行为的幽默评价
- 忠告：改进社交方式的建议
- 晒图配文：适合分享测试结果的文案

## 注意事项
- 本测试仅供娱乐，不代表专业心理评估
- 测试结果可能随心情和状态变化而不同
- 社交行为没有绝对的好坏，每种类型都有其独特之处
- 若测试结果让你感到不适，建议寻求专业心理咨询
